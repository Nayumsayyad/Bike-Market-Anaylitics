# Bike Market Analytics Dashboard

## Project Overview

This project is an interactive **Bike Market Analytics Dashboard** built using **PostgreSQL, SQL, and Power BI**. The dashboard analyzes bike market data to provide insights into pricing, mileage, GST impact, customer behavior, and value-for-money models.

The project demonstrates how business intelligence tools can be used to transform raw data into meaningful insights for decision-making.

---

## Objectives

* Analyze bike pricing across different brands and segments.
* Compare mileage and performance of bikes.
* Study GST impact on bike prices.
* Understand customer buying behavior and price sensitivity.
* Identify the most value-for-money bikes.
* Create an interactive dashboard with filters and KPIs.

---

## Dataset Information

The dataset contains information about:

* Brand
* Model
* Segment (Budget, Mid, Premium)
* Year
* Engine CC
* Mileage (km/l)
* Top Speed (km/h)
* Factory Price
* On-Road Price
* GST Amount
* GST Impact Percentage
* Overall Score
* Buyer Behaviour
* Price Sensitivity
* Value Score

---

## Tools Used

* **PostgreSQL** – Data storage and SQL queries
* **SQL** – Data analysis and business queries
* **Power BI** – Dashboard creation and visualization
* **DAX** – KPI calculations and measures

---

## Dashboard Features

### KPI Cards

* Total Models
* Average Mileage
* Average On-Road Price
* Average Overall Score
* Average Top Speed

### Interactive Filters

* Brand
* Segment
* Year
* Buyer Behaviour
* Price Sensitivity

### Visualizations

* Segment Distribution
* Average GST Amount by Brand
* Average Price by Brand
* Average Mileage by Brand
* Top Models by Value Score

---

## Key Business Insights

* KTM has the highest average on-road price.
* Hero offers the highest average mileage.
* Premium bikes are more expensive but provide better performance.
* Budget bikes provide better mileage and value for money.
* GST amount is highest for premium brands due to higher prices.
* CT100 is among the best value-for-money bikes.
